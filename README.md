# BuyFlow CRM 2.0

Site estatico do BuyFlow CRM 2.0, pronto para publicar no GitHub Pages.

## Estrutura

- `index.html`: pagina principal do site.
- `assets/styles.css`: estilos extraidos do HTML original.
- `assets/app.js`: logica da aplicacao.
- `.nojekyll`: garante que o GitHub Pages publique os arquivos sem processamento do Jekyll.

## Publicar no GitHub Pages

1. Crie um repositorio no GitHub.
2. Envie estes arquivos para a branch principal do repositorio.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, selecione `Deploy from a branch`.
5. Escolha a branch principal e a pasta `/root`.
6. Salve e aguarde o link de publicacao aparecer.
